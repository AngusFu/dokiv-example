Introduction to md2vue

You can get raw text of this document here.

Markdown is a lightweight markup language with plain text formatting syntax. It is designed so that it can be converted to HTML and many other formats using a tool by the same name. —— wikipedia

Inspiration

Inspired by nuxt, md2vue was initially aimed at transforming markdown texts in our Vue project, which were used both for documentation and demostration.

Dev Dependencies

  • marked: A markdown parser and compiler. Built for speed.
  • highlight.js: Javascript syntax highlighter.
  • prism: Javascript syntax highlighter.
  • vueify: Browserify transform for single-file Vue components
  • rollup: Next-generation ES6 module bundler.
  • buble: The blazing fast, batteries-included ES2015 compiler.

Typography & UI

  • github-markdown-css: The minimal amount of CSS to replicate the GitHub Markdown style.
  • highlight.js: Code highlighting.
  • prism: Code highlighting.
